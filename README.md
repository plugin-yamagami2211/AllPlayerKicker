# AllPlayerKicker
OP(Permission)持ち以外を全員kickするプラグイン  
バグや不具合報告は配布フォーラムまで。

## Command
/allkick <理由>  
※<理由>がないとエラーが出るのは仕様です。

## Permission
- allkick		 : allkickコマンドの使用権限
- allkick.ignore : allkickコマンドでkickされない
- allkick.* 	 : 上２つのすべて
